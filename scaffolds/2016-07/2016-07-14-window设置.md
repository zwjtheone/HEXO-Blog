# windows 常用设置

1. 先选定好自己的要设置的布局、排序..

2. 打开一个文件夹，点击组织

![](http://ww4.sinaimg.cn/large/69a9ed59gw1f5t8y0suwbj204y06qt9a.jpg)

3. 选择"文件夹和搜索"选项
![](http://ww3.sinaimg.cn/large/69a9ed59gw1f5t92vjpocj2056090mxi.jpg)

4. 选择"查看"，并应用文件夹
